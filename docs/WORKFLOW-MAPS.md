## Purpose

This document is the visual index for the reusable workflow documentation.

Use [README.md](./README.md) for the narrative system overview.
Use [REPOSITORY-MAP.md](./REPOSITORY-MAP.md) for the cross-repository dependency map.
Use [BUILD-PATHS.md](./BUILD-PATHS.md) for the build-path diagrams.

## Visual-first note

These docs are intended to be viewed in a Mermaid-capable renderer such as GitHub markdown preview or the VS Code markdown preview with Mermaid support.

If you read them as raw markdown only, the diagrams will look like source code instead of rendered visuals.
